# Unique

[English Document 英文版](README.md)

Unique是Java实现的分布式ID组件，它提供了REST APIs和Java客户端（RPC、HTTP等）方式来获取ID，并且支持Spring Boot、Spring Cloud和MyBatis。

## 安装

```bash
$ git clone https://github.com/yihleego/unique.git
$ cd unique
$ mvn clean install
```
