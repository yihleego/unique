# Unique

Unique是Java实现的分布式序列号组件，它提供了REST APIs和Java客户端（RPC、HTTP等）方式来获取序列号，支持Spring Boot、Spring Cloud和MyBatis。

## 安装

请确保您的Java版本在1.8及以上。

```bash
$ git clone https://github.com/yihleego/unique.git
$ cd unique
$ mvn clean install
```
